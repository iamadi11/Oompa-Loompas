# Decision: Shareable Deal Proposal Link
Date: 2026-04-09
Phase: Phase 1 — Deal + Payment Intelligence
Status: APPROVED

## What
A creator can generate a public, read-only URL for any deal. The link shows the deal title, value, deliverables, and payment milestones — no login required. The creator can revoke the link at any time.

## Why Now
Creators currently email screenshots or manually type deal terms when negotiating with brands. A shareable link solves the negotiation communication loop and directly reduces friction on the revenue path.

## Why Not PDF Export
Higher build cost, static snapshot, can't revoke, no live data.

## User Without This Feature
Creator screenshots the page or re-types terms into an email — 5–10 minute loop per negotiation round.

## Success Criteria
- Creator generates a link in < 3 clicks
- Brand opens the link with no login prompt
- Link shows correct deliverables and payment milestones
- Creator can revoke the link at any time
